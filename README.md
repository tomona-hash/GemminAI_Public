# GemminAI | Digital Acta Diurna

わたしたちの哲学

「言葉はAIが紡ぎ、真実は数学が守る。」

GemminAIは、古代ローマの掲示板『Acta Diurna（アクタ・ディウルナ）』の精神を現代に再定義し、情報の独占を排する完全に独立した報道のあり方を形作る試みです。わたしたちは情報を管理しません。代わりに、数学的な整合性によって情報の正当性を証明します。

3つの基本原則
情報の非独占: 将来にわたって広告を一切掲載せず、ユーザーの行動を利益に変えるビジネスモデルを拒絶します。

主語国切り替え（Perspective Switcher）: 🇺🇸 🇨🇳 🇷🇺 🇯🇵 🇺🇳 などの各視点を提供します。これは単なる翻訳ではなく、ナラティブの再構成による「知的なエンターテインメント」です。

数学的透明性: 24の属性と本文を RFC 8785 (JCS) で結晶化し、25番目の刻印として SHA-256 ハッシュを生成します。

信頼の証としての「沈黙」
もし、わたしたちが提供するデータとハッシュ値が1ビットでも一致しない場合、アプリケーションは潔く「沈黙（白紙）」を選びます。その白紙の画面こそが、偽りのない真実の証跡（エビデンス）です。
